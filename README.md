# TCopy-web

Web frontend for tcopy - Twitter Like App

## Stack

- Vite
- Typescript
- React
- React Router
- React Virtuoso
- React Hook Form
- React Icons
- Redux
- Redux Toolkit
- SASS

- [x] Global Feed
- [x] Adding Posts
